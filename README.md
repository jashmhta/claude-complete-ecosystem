## 🚀 Quick Start

### Option 1: Direct Download (Recommended)
```bash
# Download the complete ecosystem
wget https://github.com/jashmhta/claude-complete-ecosystem/archive/refs/heads/master.zip
unzip master.zip
cd claude-complete-ecosystem-master

# Run the automated deployment
./deploy-from-zip.sh
```

### Option 2: Git Clone
```bash
# Clone the repository
git clone https://github.com/jashmhta/claude-complete-ecosystem.git
cd claude-complete-ecosystem

# Run the automated deployment
./deploy-from-zip.sh
```

### Option 3: Tar Archive Download
```bash
# Download the tar archive
wget https://github.com/jashmhta/claude-complete-ecosystem/archive/refs/heads/master.tar.gz
tar -xzf master.tar.gz
cd claude-complete-ecosystem-master

# Run the automated deployment
./deploy-from-zip.sh
```

## 📋 Deployment Script Features

The `deploy-from-zip.sh` script automatically:

1. ✅ **Checks system requirements**
2. ✅ **Installs all MCP servers** (claude-flow, ruv-swarm, flow-nexus)
3. ✅ **Sets up Claude configuration files**
4. ✅ **Copies scripts and executables**
5. ✅ **Sets up documentation directory**
6. ✅ **Creates desktop shortcuts**
7. ✅ **Verifies installation success**

## 🎯 Usage Examples

### Start Claude Code with MCP Servers
```bash
# Start all MCP servers in background
./start-mcp-background.sh

# Start Claude Code
./start-claude.sh
```

### Use SPARC Methodology
```bash
# Run complete TDD workflow
npx claude-flow sparc tdd "Implement user authentication"

# Run performance optimization
npx claude-flow sparc run performance-optimization "Optimize React app"
```

### Access Specialized Agents
```bash
# Use code reviewer agent
npx claude-flow agent code-reviewer "Review this pull request"

# Use security auditor
npx claude-flow agent security-auditor "Audit this codebase"
```

## 🔧 Configuration

### Claude Configuration
The `.claude.json` file contains all necessary configurations for:
- MCP server endpoints
- Agent definitions
- Workflow templates
- Security settings

### Environment Variables
Set these environment variables for optimal performance:
```bash
export CLAUDE_MCP_SERVERS_PATH="$HOME/mcp-servers"
export CLAUDE_AGENTS_PATH="$HOME/.claude/agents"
export CLAUDE_WORKFLOWS_PATH="$HOME/.claude/workflows"
```
